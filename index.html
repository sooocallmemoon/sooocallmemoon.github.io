Clients & installation
ℹ️ Cette page récapitule les méthodes d'installation du client PRONOTE sur les différents postes de l'établissement.
🖥️

Client Windows

Installation du client lourd PRONOTE sur poste Windows (enseignants, administration).

  • 1 Télécharger le fichier d'installation depuis Index Éducation
  • 2 Lancer SetupPRONOTE.exe en administrateur
  • 3 Renseigner l'URL du serveur : —
  • 4 Valider et tester la connexion
🍎

Client macOS

Installation du client PRONOTE sur Mac (salle des professeurs, administration).

  • 1 Télécharger PRONOTE.dmg depuis Index Éducation
  • 2 Glisser l'application dans le dossier Applications
  • 3 Renseigner l'URL du serveur : —
  • 4 Autoriser l'app dans Préférences Système → Sécurité
🌐

Accès navigateur (web)

Accès sans installation via navigateur — recommandé pour élèves et parents.

  • 1 Ouvrir Chrome, Firefox ou Edge
  • 2 Naviguer vers l'URL correspondant à son profil
  • 3 Se connecter avec ses identifiants PRONOTE
URLs d'accès — Élèves — Parents — Professeurs

Application mobile

L'application PRONOTE est disponible sur iOS et Android.

  • 1 Télécharger « PRONOTE » sur l'App Store ou Google Play
  • 2 Entrer l'URL ou le code QR de connexion
  • 3 Se connecter avec ses identifiants habituels
